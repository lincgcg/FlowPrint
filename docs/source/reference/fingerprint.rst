Fingerprint
===========

A Fingerprint object holds the fingerprints as generated by FlowPrint.
These fingerprints are sets of (dst ip, dst port)-tuples and TLS certificates.
Essentially, it extends the frozenset (i.e., an unchangable set) class with methods useful for comparing, identifying, reading and storing fingerprints.
